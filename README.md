# 📊 Template de Planilha de Faturamento

Este repositório contém um modelo completo para controle de faturamento, ideal para [MEI / Pequenas Empresas / Freelancers].

## Funcionalidades
*   **Fluxo de Caixa:** Registro de entradas, controle diario.
*   **Dashboard Visual:** Gráficos automáticos de desempenho mensal e anual.
*   **Relatorios:** Relatorios com desempenho tanto anual dividido por messes e semanal, do ano todo.
*   **Controle diario:** Controle Diario de cada lançamento e total faturado no dia.

##  Como usar
1. Acesse o arquivo [FAÇA UMA COPIA]([https://www.google.com](https://docs.google.com/spreadsheets/d/19F7Za8Y6oYJh6REVrLocha_NVvUWWAOMaoLqXwwLyhQ/copy))
2. Ele vai te dar duas opções, fazer uma copia ou acessar o arquivo gs. apenas faça a copia, nesse projeto não teremos scripts.
3. Substitua os nomes na aba dados a partir da coluna D. para o nome dos vendedores. E aproveite a planilha .
